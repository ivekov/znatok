Telegram

Не настроен

Подключите Telegram-бота к вашему корпоративному помощнику.

Получите у @BotFather
Инструкция по подключению
  1. Напишите @BotFather и создайте нового бота.
  2. Скопируйте полученный токен.
  3. Вставьте его выше и нажмите «Сохранить».
  4. Бот автоматически запустится при следующем старте сервиса.

Битрикс24

Не настроен

Подключите бота к мессенджеру Битрикс24 через вебхук.

Получите в CRM → Приложения → Входящие вебхуки (раздел «Входящие вебхуки»)
Скопируйте этот URL в настройки вебхука в Битрикс24
Инструкция по подключению
  1. Создайте входящий вебхук в Битрикс24 → Приложения → Входящие вебхуки.
  2. Выберите события: ONIMBOTMESSAGEADD, ONIMCOMMANDADD.
  3. Укажите вебхук URL из поля выше.
  4. Скопируйте Access Token (последняя часть URL) и вставьте как Client Secret.
